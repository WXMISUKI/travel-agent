# 旅行规划智能体 - 简历项目介绍

## 项目概述

基于 **LangGraph** 框架和 **MiniMax 大模型** 构建的智能旅行规划助手，实现了从自然语言理解到复杂任务编排的端到端解决方案。项目采用企业级架构设计，具备智能规划、自动降级、时间上下文管理和完整审计追踪能力。

**技术难度：** ⭐⭐⭐⭐⭐
**项目周期：** 持续迭代，当前版本 v2.0.0
**代码规模：** 5000+ 行核心代码

---

## 技术栈

### 后端
- **Python 3.11+** + **FastAPI** + **LangChain** + **LangGraph**
- **Pydantic** (数据验证) + **uvicorn** (ASGI 服务器)
- **aiohttp** (异步 HTTP) + **loguru** (日志管理)

### AI/LLM
- **MiniMax M2.5** (阿里云 DashScope)
- **LangChain OpenAI** (客户端封装)
- **Prompt Engineering** (提示词工程)

### 数据源
- **Open-Meteo API** (天气数据)
- **12306 MCP** (火车票数据，MCP 协议)
- **百度搜索 API** (降级搜索)

### 前端
- **HTML5 + CSS3 + JavaScript ES6+**
- **SSE** (Server-Sent Events) 流式响应

## 核心功能

### 1. 智能对话理解
- 支持 7 类意图识别（火车票、天气、景点、交通等）
- 自动提取关键实体（出发地、目的地、日期等）
- 多轮对话上下文管理

### 2. 智能执行规划
- DAG 任务编排，管理工具依赖关系
- 自动生成执行计划，智能补全参数
- 支持条件分支和并行执行

### 3. 时间上下文管理
- 自然语言日期解析（"明天"、"下周五"等）
- 相对时间计算和时区处理
- 支持多种日期表达方式

### 4. 多数据源集成
- 天气查询、火车票查询、景点搜索
- 遵循 MCP 协议与第三方服务集成
- 支持 REST API 和 MCP 协议

### 5. 智能降级系统
- 多级降级策略，确保服务可用性
- 自动参数修复（"北京县"→"北京市"）
- 主工具失败时自动切换备用方案

### 6. 企业级审计系统
- 完整的会话追踪和执行链记录
- 性能监控和指标统计
- 结构化事件日志和错误分析

### 7. 流式响应输出
- 基于 SSE 的实时事件流
- 分步骤展示思考过程和执行状态
- 提升用户体验

## 技术亮点

### 1. LangGraph 状态机工作流
实现复杂的多步骤任务编排，支持条件分支和循环，使用状态机管理执行流程。

### 2. MCP 协议集成
遵循 Model Context Protocol 标准，实现与 12306 服务的标准化通信。

### 3. 智能参数修复
自动检测和修复错误参数，从错误信息提取修复建议，提高查询成功率。

### 4. 并发优化
分析工具依赖关系，并行执行无依赖工具，使用 asyncio 实现异步调用。

### 5. 缓存机制
基于内存的缓存实现，TTL 管理，减少重复查询，提升响应速度。

## 技术难点与解决方案

### 难点 1：复杂的依赖关系管理
**解决方案：** 使用 DAG 表示依赖关系，拓扑排序确定执行顺序，参数占位符解析实现步骤间数据传递。

### 难点 2：自然语言日期解析
**解决方案：** 实现时间上下文管理器，支持相对日期和星期表达，时区处理和日期标准化。

### 难点 3：数据源不稳定
**解决方案：** 实现多级降级策略，参数智能修复，自动错误检测和恢复。

### 难点 4：性能监控和调试
**解决方案：** 实现企业级审计日志系统，结构化事件记录，执行链追踪和性能指标统计。

### 难点 5：实时用户体验
**解决方案：** 实现 SSE 流式响应，实时推送执行状态，异步工具调用优化性能。

## 核心设计模式

1. **智能规划模式（Smart Planning Pattern）** - 基于意图识别的执行计划生成
2. **降级策略模式（Fallback Pattern）** - 多级降级机制，自动错误恢复
3. **时间上下文模式（Time Context Pattern）** - 会话级时间状态管理
4. **审计追踪模式（Audit Trail Pattern）** - 结构化事件日志，执行链追踪

---

## 技术创新点

### 1. 智能规划器（SmartPlanner）
将传统的 ReAct 模式升级为智能规划模式，提前生成完整执行计划，支持工具依赖和并行。

### 2. 时间上下文（TimeContext）
独立的时间管理模块，会话级时间状态，支持相对时间计算和多种日期表达。

### 3. 智能参数修复
自动检测和修复错误参数，提高成功率，减少用户手动干预。

### 4. 审计日志系统
企业级的完整审计追踪，支持可追溯性、性能分析和问题诊断。

## API 接口

### 基础接口
- `GET /` - 服务信息和版本
- `GET /health` - 健康检查
- `POST /chat` - 普通聊天

- `GET /audit/metrics` - 获取性能指标
