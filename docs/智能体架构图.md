## 2. TravelAgent 执行流程 (graph.py)

```mermaid
flowchart TD
    START([用户请求]) --> PARSE[解析意图<br/>_parse_intent]
    
    PARSE --> CHECK{需要哪些工具?}
    CHECK -->|天气| WEATHER[调用 get_weather]
    CHECK -->|火车票| TICKETS[调用 get_train_tickets]
    CHECK -->|景点| ATTR[调用 search_attractions]
    
    WEATHER --> FALLBACK1{工具失败?}
    TICKETS --> FALLBACK2{工具失败?}
    ATTR --> FALLBACK3{工具失败?}
    
    FALLBACK1 -->|是| BW[百度天气搜索]
    FALLBACK2 -->|是| BT[百度交通搜索]
    FALLBACK3 -->|是| BA[百度景点搜索]
    
    FALLBACK1 -->|否| RESP1
    FALLBACK2 -->|否| RESP2
    FALLBACK3 -->|否| RESP3
    
    BW --> RESP1
    BT --> RESP2
    BA --> RESP3
    
    RESP1 --> MERGE[合并结果]
    RESP2 --> MERGE
    RESP3 --> MERGE
    
    MERGE --> LLM[LLM 生成回复<br/>_make_response]
    LLM --> END([返回结果])
```

## 3. SmartPlanner 工作流 (smart_planner.py)

```mermaid
flowchart TD
    INPUT[用户查询] --> INTENT[意图识别<br/>IntentType]
    
    INTENT --> TYPE{意图类型}
    TYPE -->|TRAIN_TICKETS| TRAIN[火车票流程]
    TYPE -->|WEATHER| WEATHER[天气流程]
    TYPE -->|ATTRACTIONS| ATTR[景点流程]
    TYPE -->|TRIP_PLAN| TRIP[旅行规划流程]
    TYPE -->|CAPABILITY| CAP[能力说明]
    TYPE -->|GENERAL| GEN[通用搜索]
    
    TRAIN --> STATION1[get_station_by_city<br/>查出发站]
    STATION1 --> STATION2[get_station_by_city<br/>查目的站]
    STATION2 --> TICKET[get_train_tickets]
    
    WEATHER --> W[get_weather]
    ATTR --> A[search_attractions]
    
    TRIP --> DATE[parse_date]
    DATE --> WTRIP[get_weather]
    WTRIP --> ATRIP[search_attractions]
    ATRIP --> STRIP{有出发地?}
    STRIP -->|是| ST[get_station_by_city]
    STRIP -->|否| WSEARCH[web_search]
    ST --> WSEARCH
    
    CAP --> CAP_R[返回能力说明]
    GEN --> WSEARCH
    
    TICKET --> FALLBACK{失败?}
    W --> FALLBACK
    A --> FALLBACK
    WSEARCH --> FALLBACK
    
    FALLBACK -->|是| FALL[web_search 降级]
    FALLBACK -->|否| RESP
    
    FALL --> RESP[LLM 生成回复]
    
    RESP --> OUTPUT([返回结果])
```

## 4. LangGraph Workflow (workflow.py)

```mermaid
flowchart LR
    subgraph LangGraph["LangGraph 工作流"]
        START([__start__]) --> INTENT[意图识别<br/>intent_recognition]
        INTENT --> ENTITY[实体提取<br/>entity_extraction]
        ENTITY --> DATE[日期解析<br/>date_parsing]
        DATE --> PARAMS[参数准备<br/>prepare_params]
        
        PARAMS --> DECIDE{工具执行?}
        DECIDE -->|是| TOOL[execute_tool]
        DECIDE -->|否| RESP[直接回复]
        
        TOOL --> SUCCESS{成功?}
        SUCCESS -->|是| GEN[生成回复]
        SUCCESS -->|否| FALL[降级搜索]
        FALL --> GEN
        
        RESP --> END([__end__])
        GEN --> END
    end
```

## 6. 数据流向图

```mermaid
sequenceDiagram
    participant User as 用户
    participant API as FastAPI
    participant Agent as TravelAgent
    participant Planner as SmartPlanner
    participant Tools as Tools Layer
    participant LLM as MiniMax LLM
    participant Data as 数据源
    
    User->>API: 用户请求
    API->>Agent: 转发请求
    Agent->>Planner: 生成执行计划
    Planner->>LLM: 意图识别/实体提取
    LLM-->>Planner: 返回意图和实体
    
    Planner->>Tools: 依次执行工具
    Tools->>Data: 调用外部 API
    Data-->>Tools: 返回数据
    Tools-->>Planner: 返回工具结果
    
    Planner->>LLM: 生成回复
    LLM-->>Planner: 返回自然语言回复
    Planner-->>Agent: 返回结果
    Agent-->>API: 返回响应
    API-->>User: 流式返回结果
```
