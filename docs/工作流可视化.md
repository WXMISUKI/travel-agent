# LangGraph 工作流可视化

## 使用方法

运行以下命令生成 LangGraph 工作流图片：

```bash
cd d:\AI\AIcode\travel-agent
python -m src.agent.workflow
```

这将生成 `langgraph_workflow.png` 图片文件。

---

## 工作流架构

```
┌─────────────────────────────────────────────────────────────────┐
│                    LangGraph Workflow 流程                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  __start__                                                     │
│      │                                                         │
│      ▼                                                         │
│  ┌────────────────────┐                                         │
│  │ intent_recognition │ ─── 意图识别                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│  ┌────────────────────┐                                         │
│  │  entity_extraction  │ ─── 实体提取                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│  ┌────────────────────┐                                         │
│  │   date_parsing     │ ─── 日期解析                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│  ┌────────────────────┐                                         │
│  │  prepare_params    │ ─── 参数准备                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│     ┌──────┴──────┐                                             │
│     │ 条件边判断   │                                             │
│     └──────┬──────┘                                             │
│            │                                                     │
│            ▼                                                     │
│  ┌────────────────────┐                                         │
│  │   execute_tool     │ ─── 工具执行                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│  ┌────────────────────┐                                         │
│  │ generate_response  │ ─── 生成回复                            │
│  └─────────┬──────────┘                                         │
│            │                                                     │
│            ▼                                                     │
│      __end__                                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘

节点说明：
- intent_recognition: 识别用户意图（火车票/天气/景点/旅行规划）
- entity_extraction: 提取实体（出发地、目的地、日期等）
- date_parsing: 解析自然语言日期（明天、后天等）
- prepare_params: 准备工具调用参数
- execute_tool: 执行工具并处理降级
- generate_response: 生成最终回复
